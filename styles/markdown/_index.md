# Style Atlas — Markdown Styles

This directory documents all styles available for Markdown.

## Currently Included

### Dark Themes
- **`blackops.md`** — Industrial, terminal-grade dark theme
- **`redacted.md`** — Classified, blackout dossier theme
- **`ghost.md`** — Corrupted archive / leaked ghost archive theme

### Vibrant Themes
- **`cyberpunk.md`** — Neon-infused, retro-futuristic aesthetic with cyan and magenta
- **`vapor.md`** — Retro 80s/90s vaporwave aesthetic with pastel gradients

### Classic Themes
- **`noir.md`** — Classic film noir aesthetic with high contrast black and white

## Usage

Each markdown file contains:
1. Embedded CSS for HTML rendering
2. Style tag for meta-tracking
3. Rendering instructions
4. Design notes

For implementation details, see `styles/_meta/CONVENTIONS.md`.
